Quick Start Guide

Get up and running in 30 seconds

1

Basic Usage

Simply add this to your GitHub README.md:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME)
2

With Theme

Customize with your favorite color scheme:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME&theme=tokyonight)
3

With CP Stats

Show off your competitive programming stats:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME&leetcode=YOUR_LEETCODE_USERNAME&codeforces=YOUR_CF_USERNAME)
4

Full Customization

Combine all options for maximum impact:

![samdev-pulse](https://samdev-pulse.vercel.app/api/profile?username=YOUR_USERNAME&theme=dracula&leetcode=YOUR_LEETCODE_USERNAME&align=center)

Available Parameters

Parameter Type Default Description
username string SamXop123 Your GitHub username
theme string dark Theme: dark, light, dracula, nord, tokyonight, monokai, gruvbox, solarized, catppuccin, rose-pine, aurora, midnight-sunset, onedarkpro, material, synthwave84, forestnight, oceanicnext, emberglow, midnightneon, pasteldream
leetcode string - LeetCode username
codeforces string - Codeforces username
codechef string - CodeChef username
align string left Header alignment: left, center, right
hide_trophies boolean false Hide the achievement trophies row

Meet the Creator

Hi, I'm SamXop123—a passionate developer who believes your GitHub profile should be as unique as your code. I built samdev-pulse because I wanted a single, beautiful way to showcase everything that matters: stats, streaks, skills, and achievements.

This project combines my love for clean design, efficient APIs, and developer experience. It's fast, reliable, and will be maintained as an open-source project for the community.
